# Smart Water Level Monitoring Syste

 An IoT-based system to monitor water levels in tanks and prevent overflow using Arduino and ultrasonic sensors.

# Project Overview
Water overflow is a common issue in homes and buildings due to lack of monitoring.  
This project provides a smart solution to monitor water levels in real time and alert users when the tank is full.

# Objectives
- Monitor water level in the tank
- Prevent overflow
- Provide alert using buzzer
- Display water level on LCD
- Reduce water wastage

# Components Used
- Arduino UNO  
- Ultrasonic Sensor (HC-SR04)  
- 16x2 LCD Display (I2C)  
- Buzzer  
- Jumper Wires  
- Power Supply  

# Working Principle
- The ultrasonic sensor measures the distance between sensor and water surface
- Arduino processes the data
- Water level is displayed on LCD
- When tank is full:
  - Buzzer gives alert
  - Display shows warning
- Helps user switch OFF motor in time

# Features
- Real-time monitoring  
- Automatic alert system  
- Low cost and easy to install  
- Non-contact water level detection
